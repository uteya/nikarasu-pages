にからすとは何か（1文）

「AIで現実を変える」 — デジタルの中に閉じたAIを、カメラ・センサー・エッジデバイスを通じて現実世界と接続する会社。

事業の因果構造（ぐるぐるモデル v3.0）

にからすのビジネスは以下のループで回る。これが全戦略の土台。

プロダクト開発（WazaAI / KataribeAI）+ AI駆動開発（受託で鍛えた技術力） | v 実績・技術ナレッジ AI広報自動発信（技術ブログ / SNS / デモ動画 / AI広報キャラ） | v ブランド認知「AIで現実を変える = にからす」 | +------+------+------+ | | | | v v v v Physical AI駆動 中小DX 既存コンサル AIコンサル 開発案件 案件獲得 維持・深耕 | | | | +------+------+------+ | v 売上 + 現場知見 プロダクト開発にフィードバック --> ループ

因果構造の要点

プロダクトを作ること自体がブランドを生み、ブランドが案件を呼ぶ。 だから「プロダクト開発に時間を使う」は「営業している」のと同義。逆に、プロダクト開発が止まると発信が止まり、案件が細る。

にからすの競合戦略

ExaWizards/ABEJAはクラウドコンサル中心 → Claude Codeに食われる。McKinsey自身がAIで5,000名削減する世界。
にからすはAIにできない領域で戦う = エッジ + フィジカル。
HWとの統合 / 現場の暗黙知のデジタル化 / Privacy by Designのエッジ処理 — これらはAIツールだけでは実現できない。
